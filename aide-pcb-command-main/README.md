# AIDE-PCB Command Center

Welcome to the **AIDE-PCB** project dashboard and frontend application.

## Overview

AIDE-PCB is a comprehensive digital twin and command center for hardware debugging, visualization, and strategic AI-driven analytics.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- React Three Fiber
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to run the application locally:

```sh
# Step 1: Install dependencies
npm install

# Step 2: Start the development server
npm run dev
```

Visit the application at the local host assigned by Vite (typically `http://localhost:8081`).
